# 🚛 運転性格診断プロファイラー (Safety Drive Profiler)

## 📌 概要
本プロジェクトは、7年以上の物流現場（大型ドライバー・自販機ルートマン）で培った安全運行・データ駆動管理のドメイン知識と、Webフロントエンド技術を掛け合わせて開発した**「プロドライバーのための本音を暴く運転性格診断クイズ」**です。

一般的なクイズアプリにある「1つの正解（100点）か不正解（0点）か」という単純な2値化ロジックではなく、現場のリアルな心理や他者依存のリスクに応じた**「安全性のグラデーション（傾斜配点アルゴリズム）」**をJavaScriptで実装しています。

### 🛠️ 技術的な特徴とこだわり（事実ベース）
- **傾斜配点ロジック:** 選択肢ごとに `+2`（最適解）、`+1`（現場のやむを得ない判断）、`-1`（主観過信）、`-3`（二次災害リスク）といった多次元のスコアリング配列を保持。
- **軽量・クリーンな構成:** 外部ライブラリ（jQuery等）に依存しないバニラJavaScriptで実装し、Qiita記事や既存のWebアプリケーション（Flask/Django等）への埋め込みやすさを最優先。
- **拡張性（Data-Driven ready）:** 設問データを独立したオブジェクト配列として分離。将来的なJSON外部化や、バックエンドAPIからの非同期取得（Fetch API）へ即座にリファクタリング可能な設計。
