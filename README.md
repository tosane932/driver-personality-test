## 📌 概要
本プロジェクトは、運送・物流の現場（トラックドライバー・自販機ルートマン）で10年以上ハンドルを握ってきた経験（ドメイン知識）と、Webフロントエンドの基礎技術を掛け合わせて開発した**「運転性格診断クイズ（全50問）」**です。

一般的なクイズアプリにある「1つの正解か不正解か」という単純な2値化ロジックではなく、現場のリアルな心理や他者依存のリスクに応じた**「安全性のグラデーション（5段階の傾斜配点アルゴリズム）」**をJavaScriptで実装しています。法律上の正論だけでは割り切れない現場の葛藤やリスク許容度を、自分自身の運転感覚と照らし合わせて客観的に振り返ることができるツールです。

### 🛠️ 技術的な特徴とこだわり（事実ベース）
- **5段階の傾斜配点データ構造:** 全50問すべての選択肢に `2.0点` / `1.5点` / `1.0点` / `0.5点` / `0.0点` の5段階評価を一貫して定義し、最終的に「0点〜100点満点」で算出される加算式ロジックを組んでいます。全50問の選択肢すべてを共通の5択構造に統一し、データ構造（`quizData.js`）の対称性を意識しました。
- **軽量・クリーンな構成 & 拡張性:**
  外部ライブラリ（jQuery等）に依存しないバニラJavaScript（Vanilla JS）で実装。設問データを独立したオブジェクト配列として完全に分離しており、将来的なJSON外部化やバックエンドAPI（Flask/Django等）からの非同期取得（Fetch API）へ即座に対応可能な設計にしています。

## 📊 診断カテゴリ（全50問）

運行管理において重要視される5つのセクターに分類し、全方位のリスクを網羅しています。
1. **交差点・信号判断** (10問)：サンキュー事故、蒸発現象、ピラー死角の二段階確認など
2. **緊急車両対応** (10問)：「2連急（続く緊急車両）」の警戒、進路譲渡時の後続車プレッシャー対策など
3. **車間距離** (10問)：3秒ルールの徹底、環境変化（雨・下り坂・総重量）による物理限界の予測など
4. **悪天候** (10問)：ゲリラ豪雨、濃霧、ブラックアイスバーン、アンダーパス冠水時の迂回判断など
5. **疲労・眠気** (10問)：脳疲労サイン（一瞬の幻覚）の客観的察知、下車確認の徹底、法令遵守など
