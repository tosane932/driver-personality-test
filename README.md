# 🚛 運転性格診断プロファイラー (Safety Drive Profiler)

## [運転性格診断クイズ👈️](https://tosane932.github.io/driver-personality-quiz/)
(クリックすると運転性格診断クイズのページに移動します)

## 📌 概要
本プロジェクトは、7年以上の物流現場（大型ドライバー・自販機ルートマン）で培った安全運行・データ駆動管理のドメイン知識と、Webフロントエンド技術を掛け合わせて開発した**「プロドライバーのための本音を暴く運転性格診断クイズ（全50問）」**です。

一般的なクイズアプリにある「1つの正解か不正解か」という単純な2値化ロジックではなく、現場のリアルな心理や他者依存のリスク、物理限界（空走・制動距離、摩擦低下）に応じた**「安全性のグラデーション（5段階の傾斜配点アルゴリズム）」**をJavaScriptで実装しています。プロでも唸る難易度とリアルな現場の葛藤を網羅し、回答者自身の最新の運転傾向を客観的にプロファイリングします。

### 🛠️ 技術的な特徴とこだわり（事実ベース）
- **リカート尺度に基づく5段階傾斜配点:** 全50問すべての選択肢に `2.0点`（プロの防衛運転）、`1.5点`（セオリー順守）、`1.0点`（標準確認）、`0.5点`（リスク許容）、`0.0点`（重大リスク）の5段階評価を完全に一貫させて定義。合計値が綺麗な加算式の「0点〜100点」に収まる堅牢なデータバリデーション（境界値ガード）を実装しています。
- **浮動小数点数の丸め演算処理:**
  小数点を含むスコア（`1.5` や `0.5`）を配列の `reduce` メソッドで合算する際、JavaScript特有の浮動小数点数の計算誤差（丸め誤差）を排除するため、`Math.round(rawScore * 10) / 10` による厳密な補正ロジックを搭載。
- **シームレスなユーザー体験（F5リロード不要のUX）:**
  診断完了後のやり直し時、ページ全体を再読み込み（リロード）することなく、ボタン一発で内部データ（解答履歴・進捗インデックス）を0クリアして1問目のカードへ画面遷移する `resetQuiz` 関数を実装。1つの端末を現場で複数人に使い回して連続診断できる運用耐性を確保しています。
- **軽量・クリーンな構成 & 拡張性:**
  外部ライブラリ（jQuery等）に依存しないバニラJavaScriptで実装。設問データを独立したオブジェクト配列（`quizData.js`）として完全に分離しており、将来的なJSON外部化やバックエンドAPI（Flask/Django等）からの非同期取得（Fetch API）へ即座に対応可能な設計。

## 📊 診断カテゴリ（全50問）

プロの運行管理において重要視される5つのセクターに分類し、全方位のリスクを網羅しています。
1. **交差点・信号判断** (10問)：サンキュー事故、蒸発現象、ピラー死角の二段階確認など
2. **緊急車両対応** (10問)：「2連急（続く緊急車両）」の警戒、進路譲渡時の後続車プレッシャー対策など
3. **車間距離** (10問)：3秒ルールの徹底、環境変化（雨・下り坂・総重量）による物理限界の予測など
4. **悪天候** (10問)：ゲリラ豪雨、濃霧、ブラックアイスバーン、アンダーパス冠水時の迂回判断など
5. **疲労・眠気** (10問)：脳疲労サイン（一瞬の幻覚）の客観的察知、下車確認の徹底、法令遵守など
